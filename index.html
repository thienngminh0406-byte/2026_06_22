About

ℹ️ About JobLens

서울시 공공 채용공고 데이터를 실시간으로 수집하고 정보 품질을 정량 분석하는 HR Analytics 서비스

🎯
Why JobLens?
많은 채용공고는 직무·급여·복지·근무조건·지원방법 등 핵심 정보가 충분하지 않습니다. JobLens는 이 문제를 데이터로 수치화하고, 인사 담당자가 더 좋은 공고를 작성할 수 있도록 돕습니다.
🌐
실시간 데이터
서울시 오픈API(GetJobInfo)에서 1시간 주기로 채용공고를 자동 수집합니다. 마감된 공고는 자동으로 필터링되며, 신규 공고만 유지됩니다.
🏅
Grade Guide
S등급(90+) — 시장 최상위 공고
A등급(80+) — 정보가 매우 충실
B등급(70+) — 주요 정보 제공
C등급(55+) — 일부 보완 필요
D등급(~55) — 핵심 정보 부족
Data Pipeline
1
실시간 수집 — 서울시 오픈API(GetJobInfo)에서 1,000건씩 페이징하여 전체 채용공고 수집. 마감 공고 자동 필터링
2
1시간 캐시 — 수집된 데이터를 서버 메모리에 캐시. TTL 만료 시 백그라운드 자동 갱신 (서비스 중단 없음)
3
JobLens 스코어링 — 6개 항목(직무상세성·기업소개·급여품질·복지·근무조건·출퇴근편의) 자동 분석 및 점수·등급 산출
4
REST API 제공 — Flask 백엔드가 /api/stats, /api/jobs 등 엔드포인트를 통해 프론트엔드에 실시간 데이터 서빙
